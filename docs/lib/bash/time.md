# Time Module Reference

Time helpers from `lib/bash/time`.

## `time.epoch`
Prints the current Unix epoch timestamp (seconds since 1970-01-01 UTC).

```bash
time.epoch
```

```text
1714598400
```

## `time.iso8601`
Outputs the current UTC time in ISO 8601 format.

```bash
time.iso8601
```

```text
2024-05-01T00:00:00Z
```
